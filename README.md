# simple-go-crud

A simple CRUD application written in GOlang.

Documentation to follow: https://documenter.getpostman.com/view/13083525/UzXNUHc2

To load the app:

1. Clone the repo
2. Run go get
3. cd cmd/main
4. go build
5. go run main.go
6. server is available on local host via http://127.0.0.1:5000
